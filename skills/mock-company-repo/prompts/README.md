# Generated prompts

This folder holds **prompt files generated by a `/mock-company-repo` run** for a specific company. It is intentionally empty in the plugin repo — do not commit historical example prompts here.

When the skill runs:
1. Research the company and write `prompts/<slug>.md` (the build spec).
2. In generate-and-build mode, implement that spec into **demo-monorepo**, verify with **computer use** (screen recording of the running app), and open a **PR**.

Keep this directory for new generated specs only; clean up or leave uncommitted ephemeral prompts as needed.
